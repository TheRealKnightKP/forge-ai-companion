# Forge AI Companion

**Voice-activated AI companion for your game.**

Forge listens to your voice, responds in character, executes macros, remembers context, and reacts to what's happening in your game — in real time.

## What Forge Does

- 🎙️ Voice recognition (Whisper, runs locally)
- 🤖 AI responses in character (Groq online or Gemma4 local/offline)
- ⚡ Voice-activated macros — any keybind combination
- 🧠 Session and permanent memory
- 📋 Voice logging to file
- 🎮 Star Citizen Game.log event reactions
- 👁️ HUD vision — say "forge look at [region]" for AI to analyze your screen
- 🔇 Tool-only mode — offline, instant, no AI needed
- 🔕 TTS toggle — responses in log only if you prefer silence

## Requirements

- Windows 10/11 64-bit
- Free Groq API key → [console.groq.com/keys](https://console.groq.com/keys)
- For Local mode: [Ollama](https://ollama.com) + `ollama pull gemma4:e4b`
